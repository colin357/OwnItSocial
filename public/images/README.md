# Image Upload Instructions

Please upload the following images to this directory (`public/images/`) with these exact filenames:

## Required Images:

1. **hero-business.png**
   - The business collage image (laptop with people on phone/in meetings)
   - Used in: Hero section (top of homepage)
   - Recommended size: 1200x1000px or larger
   - Format: PNG

2. **colin-portrait.webp**
   - Colin's portrait photo (casual attire, wooden background)
   - Used in: About section
   - Recommended size: 800x800px or larger
   - Format: WebP

3. **video-production.webp**
   - Image of video/content production (camera on tripod)
   - Used in: Value proposition section (top left box)
   - Recommended size: 600x400px or larger
   - Format: WebP

4. **chatgpt-phone.webp**
   - Image of phone showing ChatGPT interface
   - Used in: Value proposition section (top right box)
   - Recommended size: 600x400px or larger
   - Format: WebP

5. **success-team.webp**
   - Image of business people running/celebrating success
   - Used in: Value proposition section (bottom box)
   - Recommended size: 1200x400px or larger
   - Format: WebP

## Image Guidelines:

- **Format**: PNG for hero-business, WebP for all others
- **Quality**: High resolution, optimized for web
- **Aspect Ratios**: Try to maintain the recommended sizes for best display
- **File Size**: Aim for under 500KB per image after optimization

## After Uploading:

Once all images are uploaded, run:
```bash
npm run build
```

This will verify that all images are properly referenced and optimized by Next.js.
